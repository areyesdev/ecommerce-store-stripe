# Ecommerce Store Stripe

This is an ecommerce created for testing and developed in a course taken from [Egghead](https://egghead.io/)
## Starting

First, install dependencies

```bash
npm install
# or
yarn install
```
Then run the server

```bash
npm run dev
# or
yarn dev
```
